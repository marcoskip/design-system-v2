Tela de Listagem

Quando usar: para qualquer tela de listagem de itens com filtros, KPIs no topo, ações em massa e tabela de resultados (gerenciador de preços, listagem de negociações, lista de produtos, etc.).
Quando NÃO usar: para tela de detalhe/edição de um único item, use o template Detalhe. Para tela de KPIs sem listagem grande, use o template Dashboard.

Exemplo: Gerenciador de Preços. Combina Header, Sidebar, Title bar, Big Numbers, Filtros, Cabeçalho de resultados, Grid (com células especiais) e Paginação.

Componentes do DS usados: Header · Sidebar · Title bar · Big number card · Filtro chips · Filtro dropdowns · Cabeçalho de resultados · Botões compostos · Aplicar preço · Grid · Células especiais · Paginação
