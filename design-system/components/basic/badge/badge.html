Variations

Badge possui duas variações principais: basic e complete. As cores do badge podem ser alteradas conforme a necessidade na interface, porém deve ser respeitado nossas cores semânticas.

Basic

Essa variação não possui conteúdo e tem apenas um único tamanho.

filled
ghost

Complete

Diferentemente do basic, essa variação deve sempre estar acompanhado de uma ou mais possibilidades de conteúdo: ícone, texto e/ou contador numérico.

info icon
tag texttext
99counter
info tag text 99 icon + text + counter

Sizes

O badge, na variação complete, possui dois tamanhos: medium e small sendo o tamanho small o utilizado com maior frequência.
